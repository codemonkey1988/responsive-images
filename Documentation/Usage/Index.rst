==========
How to use
==========

To make this extension working, it needs rendering variants.
These rendering variants are configured using TypoScript.
Default configurations are provides by this extension.
See :ref:`configuration` how to use these.

When variants are provided, they needs to be rendered
in frontend. The :ref:`rendering` for examples.
